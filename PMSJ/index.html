已知数列{a[n]}满足a[1]=2，a[n+1]=2( a[n]^2 )，请回答下列问题： （1）设b[n]=log[2]( a[n] )，求b[n+1]关于b[n]的关系式（即求解b[n]的递推公式）； （2）求数列a[n]的通项公式。 解： (1) 显然对于任意正整数n均有a[n]＞0（严格论证可用数学归纳法证明）。 则 b[n] = log[2](a[n])，b[n+1] = log[2]( 2*a[n]^2 ) = 2*log[2](a[n]*√2) = 2*( 1/2 + log[2](a[n]) ) = 2(b[n] + 1/2) 即b[n+1] = 2b[n] + 1 (2) （思路：构造等比数列以求解b[n]的通项公式，进而求解a[n]的通项公式） 由（1）中结论，等式两边加一常数k得 b[n+1] + k = 2b[n] + 1 +k = 2(b[n] + 1/2 + k/2) 即 k = k/2 + 1/2 ，解得k=1. 从而( b[n+1] + 1)/( b[n] + 1) = 2，又因为b[1] + 1 = log[2]( a[1] ) + 1 =2 因此数列{b[n] + 1}是首项为2，公比为2的等比数列。 则b[n] + 1 = log[2]( a[n] ) + 1 = 2*2^(n-1) = 2^n，移项后等式两边取2的指数得通项公式 a[n]=2^(2^n-1)
已知数列{a[n]}满足a[1]=2，a[n+1]=2( a[n]^2 )，请回答下列问题：
（1）设b[n]=log[2]( a[n] )，求b[n+1]关于b[n]的关系式（即求解b[n]的递推公式）；
（2）求数列a[n]的通项公式。

解：
(1) 显然对于任意正整数n均有a[n]＞0（严格论证可用数学归纳法证明）。
则 b[n] = log[2](a[n])，
b[n+1] = log[2]( 2*a[n]^2 )
= 2*log[2](a[n]*√2)
= 2*( 1/2 + log[2](a[n]) )
= 2(b[n] + 1/2)
即b[n+1] = 2b[n] + 1
(2) （思路：构造等比数列以求解b[n]的通项公式，进而求解a[n]的通项公式）
由（1）中结论，等式两边加一常数k得
b[n+1] + k = 2b[n] + 1 +k
= 2(b[n] + 1/2 + k/2)
即 k = k/2 + 1/2 ，解得k=1.
从而( b[n+1] + 1)/( b[n] + 1) = 2，
又因为b[1] + 1 = log[2]( a[1] ) + 1 =2
因此数列{b[n] + 1}是首项为2，公比为2的等比数列。
则b[n] + 1 = log[2]( a[n] ) + 1 
= 2*2^(n-1) = 2^n，
移项后等式两边取2的指数得通项公式
a[n]=2^(2^n-1)
